### What I learned

CSS Grid - using grid-templates-columns and setting each section using the grid-column style. 


```css

main {
        grid-template-columns: 1fr 1.5fr;
        text-align: left;
    }

    h1 {
        margin: 0;
    }

    /* grid layout */
    .intro {
        grid-column: 1/2;
    }

    .reviews {
        grid-column: 2/3;
        align-self:center;
    }

    .quotes {
        grid-column: 1/-1;
    }

```
